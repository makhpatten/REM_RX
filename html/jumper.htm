SB70LC adapter board layout of jumpers and connectors

DC Voltage Source
Jumper Setting Signal Description
JP2 [1-2] Supply power to the device through the USB connector (J20)
JP2 [2-3] Supply power to the device through the external power jack (J9) or contact terminal (J10)

UART 0 RS-232 via USB (J20)
Jumper Setting Signal Description
JP3 [1-2] UART 0 RX (required)
JP4 [1-2] UART 0 CTS (optional)

UART 0 RS-232 via DB9 (J1)
Jumper Setting Signal Description
JP3 [2-3] UART 0 RX (required)
JP4 [2-3] UART 0 CTS (optional)

UART 1 RS-232 via DB9 (J2)
Jumper Setting Signal Description
JP7 [1-2] UART 1 RX (required)

UART 1 RS-485 Full-duplex via 5-pin Header (J4)
Jumper Setting Signal Description
JP6 [Open] Enable RS-485 full-duplex receiver (required)
JP7 [2-3] UART 1 RX (required)
JP8 [Open] Enable echo (required); receiver always stays active

UART 1 RS-485 Half-duplex via 5-pin Header (J4)
Jumper Setting Signal Description
JP6 [Close] Enable RS-485 half-duplex receiver (required)
JP7 [2-3] UART 1 RX (required)
JP8 [Close/Open] Disable echo (close) or enable echo (open); when echo is disabled, receiver is off only when transmitter is sending

UART 1 RS-485 Signal Polarity (J4)
Jumper Setting Signal Description
JP5 [Open] Normal polarity of UART 1 RS-485 +/- signals; see below for configuration of signals on J4
JP5 [Close] Reverse polarity of UART 1 RS-485 +/- signals; see below for configuration of signals on J4

UART 0 and 1 RS-232 DB9 Connector Pin-out (J1 and J2)
Pin Number Signal Description
1 Not connected
2 Transmit data (TXD)
3 Receive data (RXD)
4 Short to pin 6
5 Ground
6 Short to pin 4
7 Clear to send (CTS)
8 Request to send (RTS)
9 Not connected

Normal Polarity Reverse Polarity
